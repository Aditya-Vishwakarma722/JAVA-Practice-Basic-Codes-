# JAVA-Practice-Basic-Codes-
This repository contains a collection of simple JAVA programs that I wrote during my Second year of learning programming. These codes are designed for beginners who want to understand the fundamentals of JAVA in the easiest way possible.

The repository covers various basic topics, such as loops, conditions, functions, arrays, class, methods and more. Each program is written in a straightforward manner, making it perfect for anyone just starting JAVA.

Feel free to explore, learn, and even modify the code to suit your own learning journey!
